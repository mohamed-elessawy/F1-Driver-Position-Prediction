# F1 Head-to-Head Driver Matchup Dashboard

A Formula 1 analytics dashboard built with Dash and Plotly for comparing driver performance and predicting race outcomes.

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Dash](https://img.shields.io/badge/Dash-2.9+-green.svg)

### App Link Hosted on Plotly Cloud: [F1-Dashboard](https://240db74e-ef70-41a5-98b5-9f1373455f4f.plotly.app)
## Features

- **Head-to-Head Comparison**: Compare two F1 drivers across multiple metrics
  - Average finishing position
  - Total career points
  - DNF (Did Not Finish) rates

- **Race Predictor**: Predict which driver finishes ahead using a binary Random Forest classifier
  - Select circuit, drivers, constructors, and grid positions
  - Trained on hybrid era data (2014–2024)
  - Returns winner and model confidence percentage

- **Top Drivers**: View top N drivers by race wins (configurable)

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mohamed-elessawy/F1-Driver-Position-Prediction.git
   cd F1-Driver-Position-Prediction
   ```

2. **Create virtual environment** (optional but recommended)
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate     # Windows
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Download F1 data**

   Download the dataset from [Kaggle — Formula 1 World Championship](https://www.kaggle.com/datasets/rohanrao/formula-1-world-championship-1950-2020) and place the CSV files in the `data/` folder.

5. **Train the model**

   Open and run `notebooks/01_model_training.ipynb` to generate:
   - `data/f1_cleaned.csv`
   - `data/drivers_lookup.csv`, `constructors_lookup.csv`, `circuits_lookup.csv`
   - `models/rf_h2h.pkl`
   - `models/model_meta.pkl`

6. **Run the dashboard**
   ```bash
   python app.py
   ```

   Open http://127.0.0.1:8050 in your browser.

## Usage

### Tab 1: H2H Stats
- Select two different drivers from the dropdowns
- View comparison charts for average finish position, total points, and DNF rates

### Tab 2: Race Predictor
- Select a circuit, two drivers, their constructors, and grid positions
- View the predicted winner and model confidence

### Tab 3: Top Drivers
- Shows the top 3–10 drivers by race wins (2014–2024)

## Power BI Dashboard

The `powerbi/` folder contains a multi-page Power BI dashboard built for deep historical analysis of Formula 1. It uses custom DAX measures, a purpose-built aggregation table, and a managed data model with defined relationships across all source tables. The color palette is aligned with F1's brand identity throughout.

**Pages:**

- **Overview** — High-level historical summary with global stats (total races, drivers, constructors), constructor win distributions by nationality, and interactive line charts with dynamic map visuals tracking constructor points over time.

- **Driver Dashboard** — Individual driver deep-dive with dynamic metrics: active seasons, win ratio, podium count, and points trajectory across different seasons.

- **Constructors & Circuits — Reliability Analysis** — Pie charts breaking down DNF causes (engine failures, accidents, collisions, etc.) that cross-filter dynamically to reveal which circuits are most problematic for each constructor. Includes SVG track layout visuals.
